Change log
Version 0.4 (Alpha release)
  • Bug fixes
  • Better unit-test coverage
  • Support for schema-enabled processing
  • Schema parser for EXI encoded XML Schemes
Features:
  • EXI Cookie
  • Options Presence Bit
  • Strict mode
  • Fragment
  • Preserve.prefixes
  • SchemaId
  • ValueMaxLength
  • ValuePartitionCapacity
Known issues:
  • Only bit-packed alignment. Byte-alignment and pre-compression are not supported
  • Compression is not supported
  • Preserving lexicalValues is not supported
  • Random access through selfContained option is not supported
  • Datatype Representation Map is not supported
  • Some exotic XML schema constructs are not implemented yet (e.g. string pattern facets for Restricted Character Sets, Lists, Unions)
